## Visão Geral

Este projeto foi criado para padronizar e agilizar o processo de feedback de QA dentro da equipe. Você insere o nome do usuário (hunter), marca os pontos a serem corrigidos, e a ferramenta gera uma mensagem bem formulada pronta para envio.

---

## Funcionalidades

- Inserção automática do nome completo do hunter  
- Seleção de diversos pontos de correção predefinidos  
- Adição de pontos personalizados, caso algum detalhe importante não esteja listado  
- Geração automática da mensagem final  
- Botão de copiar para facilitar uso imediato  

---

## Como Usar

### 1. Adicionar nome do hunter  
Digite o nome completo do hunter para que ele seja mencionado automaticamente na mensagem final.

### 2. Selecionar pontos predefinidos  
Marque os checkboxes correspondentes aos erros ou ajustes identificados.

### 3. Incluir pontos personalizados  
Se houver algum ponto específico que não esteja na lista, você pode adicioná-lo manualmente.  

### 4. Gerar e copiar mensagem  
Depois de selecionados os pontos, clique em “Gerar mensagem” e use o botão de “Copiar” para levar a mensagem ao clipboard.
